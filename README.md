# VALIDATION-FORM-USING-HTML
A simple and responsive web-based form built with HTML that collects user input and performs basic field validation. Designed to demonstrate form structure, input types, required fields, and clean user interface practices for beginners in front-end development.

## Features
- Name input
- Email validation
- Password field
- Required fields
- Clean design

## Technologies Used
- HTML
- CSS
- JavaScript

## How to Run
Open index.html in browser.
